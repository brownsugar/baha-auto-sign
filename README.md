## 立即安裝

支援主流 Chromium 核心瀏覽器如 Google Chrome、Microsoft Edge、Vivaldi 以及 Brave，暫不支援 Firefox。

## 開發指令

```bash
# install dependencies
$ yarn

# serve with hot reload
$ yarn dev

# build for production and generate zip file
$ yarn build

# lint the TypeScript files with ESLint
$ yarn lint
```
